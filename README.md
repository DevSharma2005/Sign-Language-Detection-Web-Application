# Sign-Language-Detection-Web-Application
1.Project Title
Sign Language detection Web Application
A real-time web application that uses computer vision to translate hand gestures into text/speech

2.About the project
-> Problem statement: This web application is build to remove the communication gap between general people and deaf community by converting  real-time sign's into text/speech.
This web application bridges the gap between general people and deaf community.

-> Solution: The application uses a camera feed to reconize gestures.

-> Key Features:
   Real-time detection.
   Support for specific alphabets(eg. ASL) or custom gestures
   Text-to-speech integration

3.Tech Stack
-> Frontend: Flask,HTML5,CSS3
-> Computer Vision: MediaPipe
-> Hand Tracking: OpenCV
-> ML Frameworks: TensorFlow, NumPy, Scikit-Learn
-> Programming Languages: Python
-> Text-to-speech Library: Pyttsx

4.Prerequisites
-> Python: 3.9 or higher
-> development Environment: VS Code
-> Libraries: mediapipe, opencv, tensorflow,
              flask and flask-socketio, pyttsx3, numpy

5.Environment Conditions
Lighting: Well-lit to capture gestures more prominently    and accurately
Background: Plain or neutral background to prevent false detection
